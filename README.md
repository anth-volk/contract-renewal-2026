# Contract Renewal 2026

A presentation slideshow for contract renewal discussion.

## Installation

```bash
npm install
```

## Running

Start the development server:

```bash
npm run dev
```

Then open http://localhost:5173 in your browser.

## Navigation

- Press `→` or `Space` to advance to the next slide
- Press `←` to go back to the previous slide

## Building

To build for production:

```bash
npm run build
```

The output will be in the `dist` directory.
